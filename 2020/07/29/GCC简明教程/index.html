    GCC简明教程-1(概述与GCC安装)
    event 发布于: 2020-07-29 event_available 更新于: 2020-07-29 folder 分类于: C++程序设计 visibility 阅读次数: 

    概述

    关于这篇教程

    这是一个对GNU/GCC(与G++)套件的简单介绍
    在阅读本教程之前，你需要对C++语法与命令行操作有一个基本的认识。
    你可以点击 这里(菜鸟教程) 来快速入门。




    什么是GNU/GCC

    GNU/GCC是一款 开源 的多语言编译器，属于GNU计划
    她支持C，C++，Objective-C，Java等语言的编译。且能够在Linux，Windows，Unix，MacOS，Android等系统上运行。




    现在的GCC

    不得不承认，2020年的今天，Clang，MSVC等编译器渐渐取代了GCC的位置。
    但GCC基于GPL协议发布，拥有比MSVC更高的自由度与跨平台性。
    同时，对于新手入门，GCC依然是首选。




    不同系统下的GNU/GCC

    在Linux系统下，可以使用原生GCC。
    在Window系统下，可以使用MinGW来构建GCC环境。




    把GCC搬进电脑

    “工欲善其事，必先利其器。”

    Linux

    在Ubuntu等Linux发行版上，GCC已预装于系统中。
    在终端中输入以下命令验证是否拥有GCC
    $ gcc -v

    Windows

    在Windows系统下，使用MinGW（Minimalist GNU for Windows）搭建GCC环境。
    你可以在这里下载MinGW

    在CMD中使用以下命令以验证安装
    gcc -v
